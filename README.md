# ROI Calculator

**A simple Python app for World of Warcraft that calculates crafting cost, disenchant value, and ROI of an item, helping players determine if it’s worth crafting to make gold.**

---

## Features
- Input crafting material quantities and prices.
- Input disenchant chances and prices.
- Calculates:
  - Total crafting cost
  - Total disenchant value
  - ROI (Return on Investment) percentage
- Easy-to-use GUI built with Python Tkinter.
- Generates actionable info for gold-making decisions.

## How to Use
1. Download the repository or the EXE file.
2. If using `.exe`, double-click `ROI Calculator.exe`.
3. Enter the quantities and prices for your crafting materials.
4. Enter the chance and price for disenchant drops.
5. Click **Calculate** to see crafting cost, disenchant total, and ROI (%).
6. Positive numbers only — 0 is valid.

---

## Requirements
- Windows PC
- Python 3.x (if running `.py` version)
- Tkinter library (comes pre-installed with Python)
